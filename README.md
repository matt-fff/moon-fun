Messing around with Moon.

### File Structure

- The main HTML page is located in `index.html`
- Styles are in `css/styles.css`
- All javascript code is in `js/scripts.js`

### Running

```sh
# Install dependencies
$ npm install

# Run Dev Server
$ npm run dev

# Build for Production
$ npm run build
```
